**CliMax** is a tool for exploring foil section design space and optimising sections.

Some of the major features:
- xfoil based
- Azure cloud based
- Evolutionnary algorithm based ( NSGA II and NSGA III, PSO... )
- multiobjectives



CliMax's development started in 2018
